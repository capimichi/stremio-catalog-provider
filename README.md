# stremio-catalog-provider

Un repository dedicato all'indicizzazione personalizzata (custom indexing) di serie TV e film, finalizzato a esporli come flussi/cataloghi per [Stremio](https://www.stremio.com/).

## Caratteristiche principali (Pianificate)
- **Indicizzazione Custom**: Parsing e catalogazione flessibile di film e serie TV da sorgenti personalizzate.
- **Integrazione Stremio**: Server addon compatibile con le API dei cataloghi e flussi di Stremio.
- **Filtri Avanzati**: Possibilità di filtrare per lingua, risoluzione, sorgente e metadati aggiuntivi.

## Architettura
L'addon seguirà il protocollo ufficiale degli Addon di Stremio, esponendo un manifesto (`manifest.json`) e gli endpoint dedicati per cataloghi e stream.
